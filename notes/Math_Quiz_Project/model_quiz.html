១. នៅក្នុងប្លង់ទ្វេវិមាត្រ (លំហ $\mathbb{R}^2$) តើវ៉ិចទ័រមួយត្រូវបានតំណាងដោយអ្វី?

៤. សម្រាប់គ្រប់វ៉ិចទ័រ $\vec{v}$ នៅក្នុងលំហវ៉ិចទ័រ តើផលបូក $\vec{v} + \vec{0}$ ស្មើនឹងអ្វី?




៥. តើលក្ខខណ្ឌចាំបាច់ទី១ ដែលសំណុំមួយត្រូវមាន ដើម្បីជា "លំហរងវ៉ិចទ័រ" គឺជាអ្វី?




៦. តើមួយណាខាងក្រោមតំណាងឱ្យ "លំហរង" (Subspace) នៅក្នុង $\mathbb{R}^2$?




៧. តើវ៉ិចទ័រផ្ទុយ (Additive Inverse) នៃវ៉ិចទ័រ $\vec{u} = (5, -3)$ គឺអ្វី?




៨. ការបូកវ៉ិចទ័រមានលក្ខណៈត្រឡប់។ តើសមភាពមួយណាបង្ហាញពីលក្ខណៈនេះ?




៩. តើទម្រង់ទូទៅនៃបន្សំលីនេអ៊ែររវាងវ៉ិចទ័រ $\vec{u}$ និង $\vec{v}$ គឺអ្វី?




១០. ប្រសិនបើគេយកវ៉ិចទ័រ $\vec{w}$ បូកជាមួយនឹងវ៉ិចទ័រផ្ទុយរបស់វា $(-\vec{w})$ តើលទ្ធផលគឺអ្វី?




១១. តើសមីការប្លង់ $ax + by + cz = d$ មួយណាទើបអាចចាត់ទុកជា "លំហរង" នៃ $\mathbb{R}^3$ បាន?




១២. តើសំណុំមួយណាដែលជាលំហរង (Vector Subspace) ពិតប្រាកដក្នុង $\mathbb{R}^3$?




១៣. តើប្រជុំនៃប្លង់ពីរ $P_1 \cap P_2$ (កាត់តាមគល់តម្រុយ) តំណាងឱ្យអ្វី?




១៤. តើប្រជុំនៃបន្ទាត់ពីរ $L_1 \cup L_2$ (កាត់តាមគល់តម្រុយ) ជាលំហរងនៃ $\mathbb{R}^3$ ដែរឬទេ?




១៥. តើទម្រង់ $S = \{ t(a, b, c) \mid t \in \mathbb{R} \}$ មានន័យដូចម្តេច?




១៦. តើសំណុំ $S = \{ (x, y, z) \mid x + y + z = 0 \text{ and } x - y = 0 \}$ តំណាងឱ្យអ្វី?




១៧. តើលំហរងក្នុង $\mathbb{R}^3$ អាចមានវិមាត្រប៉ុន្មានខ្លះ?

១៩. តើអ្វីជាលក្ខណៈខុសគ្នាសំខាន់រវាង "លំហអាហ្វីន" និង "លំហវ៉ិចទ័រ"?




២០. ប្រសិនបើគេយកចំណុចមួយដកចំណុចមួយទៀតក្នុងលំហអាហ្វីន ($A - B$) តើបានអ្វី?




២១. ក្នុងលំហអាហ្វីន កន្សោម $\alpha A + \beta B$ តំណាងឱ្យចំណុចថ្មី លុះត្រាតែលក្ខខណ្ឌអ្វីផ្ទៀងផ្ទាត់?




២២. លំហរងអាហ្វីនមួយត្រូវបានកំណត់ដោយធាតុសំខាន់ពីរ។ តើធាតុទាំងពីរនោះគឺជាអ្វី?




២៣. តើសមីការប៉ារ៉ាម៉ែត្រនៃបន្ទាត់អាហ្វីន កាត់ចំណុច $A$ និងមានវ៉ិចទ័រប្រាប់ទិស $\vec{u}$ គឺអ្វី?




២៤. ឧបមាថាចំណុច ៣ ផ្សេងគ្នា $A, B, C$ មិនរត់ត្រង់ជួរ។ តើលំហរងអាហ្វីនដែលបង្កើតដោយចំណុចទាំងបីនេះមានរូបរាងអ្វី?




២៥. នៅក្នុងលំហអាហ្វីន $\mathbb{R}^3$, ក្រៅពីកាត់គ្នា ស្របគ្នា និងត្រួតស៊ីគ្នា តើបន្ទាត់ពីរអាចមានស្ថានភាពអ្វីទៀត?




២៦. ចំណុចកណ្តាល $I$ នៃអង្កត់ $[AB]$ ក្នុងលំហអាហ្វីន អាចសរសេរបានជាកន្សោមណា?




២៧. តើលក្ខខណ្ឌអ្វីដែលធ្វើឱ្យប្លង់អាហ្វីន $P_1$ និង $P_2$ ស្របគ្នាដាច់ខាត (Strictly parallel)?

៣០. ក្នុងប្លង់អាហ្វីន $\mathbb{R}^2$ តើអ្វីជាភាពខុសគ្នាសំខាន់រវាង "ចំណុច" និង "វ៉ិចទ័រ"?

៣៦. ក្នុង $\mathbb{R}^2$ បន្ទាត់ពីរមិនស្របគ្នា កាត់គ្នានៅ?
