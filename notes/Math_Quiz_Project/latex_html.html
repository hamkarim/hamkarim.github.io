\section{មូលដ្ឋានគ្រឹះនៃទ្រឹស្ដីសំណុំ} \textbf{សំណុំ} $X$ គឺជាបណ្ដុំនៃវត្ថុខុសៗគ្នា ហៅថា \emph{សមាជិក/ធាតុ} របស់ $X$ ហើយ គេសរសេរ $x\in X$ មានន័យថា “$x$ គឺជាធាតុរបស់ $X$”។ នៅក្នុងមេរៀននេះ យើងសន្មតថាសំណុំមួយ ត្រូវបានកំណត់ច្បាស់លាស់ និងប្រើប្រាស់ទ្រឹស្ដីបែបបុរាណ (naïve set theory)។ \subsection{ប្រមាណវិធីលើសំណុំ} ឧបមាថា $A,B,C$ ជាសំណុំ។ យើងកំណត់៖ \begin{itemize} \item \textbf{ប្រជុំ (Union)}: \[ A\cup B=\{x\mid x\in A \text{ ឬ } x\in B\}. \] \item \textbf{ប្រសព្វ (Intersection)}: \[ A\cap B=\{x\mid x\in A \text{ និង } x\in B\}. \] \item \textbf{ផលសង (Difference)}: \[ A\setminus B =\{x\mid x\in A \text{ និង } x\notin B\}. \] \item \textbf{សំណុំបំពេញ (Complement)} នៃ $A$ ក្នុងសកល $U$: \[ A^{c}=U\setminus A. \] \end{itemize} សំណុំទទេត្រូវបានតាងដោយ $\varnothing$។ សំណុំ $X$ ត្រូវបានហៅថា \emph{កំណត់} (finite) ប្រសិនបើមានអនុគមន៍ប៊ីហ្សិកទីវ (bijection) រវាង $X$ និងចន្លោះចំនួនគត់ $\{1,2,\dots,n\}$ បើមិនដូច្នោះទេ វាជាសំណុំអនន្ត (infinite)។ \subsection{គូមានលំដាប់} ចំពោះសមាជិក $x,y$, \textbf{គូមានលំដាប់} $(x,y)$ ត្រូវបានកំណត់ជាសំណុំ $\{\{x\},\{x,y\}\}$ (ការសាងសង់តាមបែប Kuratowski)។ និយមន័យនេះធានាថា $(x,y)=(u,v)$ លុះតែ $x=u$ និង $y=v$។ ប្រសិនបើមានសំណុំ $A,B$ នោះ \textbf{ផលគុណខដេស៊ីយ៉ន} របស់វាគឺ៖ \[ A\times B=\{(a,b)\mid a\in A,\;b\in B\}. \]
